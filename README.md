# Usage
<!-- usage -->
```sh-session
$ npm install -g qbsync-cli
$ qbsync COMMAND
running command...
$ qbsync (-v|--version|version)
qbsync-cli/0.0.1 darwin-x64 node-v12.18.3
$ qbsync --help [COMMAND]
USAGE
  $ qbsync COMMAND
...
```
